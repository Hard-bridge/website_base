1. pip install django - установка фремворка django

2. django-admin startproject название_проекта) - создание проекта
3. cd electronic - переход в указанную папку
4. python manage.py startapp название_приложения - создание приложения
5. Переходим в файл settings.py
-INSTALLED_APPS - подключаем приложение shop
-меняем язык
-пишем хост 127.0.0.1
5. Делаем миграцию (импорт моделей в БД)
python manage.py makemigrations - проверка на корректность данных
python manage.py migrate - миграция моделей в базу данных
6. python manage.py createsuperuser - создание суперпользователя (админа)

8. Подключение urls из приложения (shop)
Заходим в папку с названием проекта (electronic) открываем urls
from django.contrib import admin
from django.urls import path, include

urlpatterns = [ #список в котором прописываем маршруты
    path('admin/', admin.site.urls),
    path('', include('shop.urls'))  с помощью ф-ции include подключаем файл urls из приложения shop для того чтобы в самом приложении прописывать маршруты
]

Шаги создания страницы на сайте
1. Создаем маршрут в urls.py
2. Создание ф-ции в views.py (определение ф-ций)
3. Создание шаблонов в папке templates в папке с названием приложения (shop)


основные HTNL (структура страницы) теги
<p></p> - абзац
<h1></h1>...<h6></h6> - заголовки
<img src='путь к изображению'> - вывод изображений
<b></b> - задает текст жирным
<i></i> - курсив
<a href='ссылка'>Текст ссылки</a>
<ul> маркированный список
    <li></li> - эл-ты маркированного списка
</ul>
<ol> нумерованный список
    <li></li>
</ol>

Формы (форма регистрации, форма авторизации, форма поиска)
Элементы формы (для ввода логина, пароля, checkbox, radio)
<form>
    <input type='text'>
    <input type='password'>
    <input type='email'>

    <button>Зарегистрироваться</button>
